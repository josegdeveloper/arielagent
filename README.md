# 🤖 ARIEL — Advanced Reasoning & Intelligent Execution Layer

An Artificial Intelligence agent that controls your PC using natural language. It uses Claude (Anthropic) to reason, plan, and execute automated tasks.

## Features

- **Full PC control**: mouse, keyboard, screenshots, file management, system commands.
- **Dual memory**: short-term (session) and long-term (persistent across sessions).
- **Semantic search**: finds items in memory by meaning, not just exact words.
- **Constitution (laws.json)**: rules and prohibitions the agent never violates.
- **Dual logging**: human-readable `.log` + structured `.json` for analysis.
- **Prompt Caching**: reduces costs by reusing the system prompt.
